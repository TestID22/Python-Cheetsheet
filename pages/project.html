Building a project
1) How to add a library using pip and requirements.txt

    pip -m install package_name установить новую либу

    pip list показать спискок библиотек

    py -m pip list

    py -m pip search "query"

    pip freze > requirements.txt

    pip install -r requirements.txt

constraints.txt file is used to specify upper or lower bounds for package versions.

- py -m pip install -c constraints.txt

- py -m pip install -c constraints.txt requests numpy

#constraints.txt

# Django>=3.0,

# 4.0 requests==2.26.0

# numpy<=1.21.0

2) How to install and configure python interpreter PyCharm
2) When creating a new project, it will ask for the Python interpreter.

You can configure it from File > Settings > Project: "project_name" > Python Interpreter

VENV: это изолированное цифровое пространство, в котором вы можете установить свои собственные версии Python и пакеты, отличные от тех, которые установлены в системе. Используется для изоляции зависимостей:

#myenv - is your env's name

python -m venv myenv

# to activate env:

\venv\scripts\activate

3) How to Uninstall a package

python -m pip uninstall some_package

python -m pip list --outdated - display a list of packages that have a newer version available
